## Development

- Install Homebrew on macOS
- Install node
    ```bash
    brew install node@6
    ```
- Checkout this repo
    ```bash
    git clone git@github.com:rijn/i-want-a-plus
    ```
- Install depedencies
    ```bash
    sudo npm i -g grunt grunt-cli
    npm i
    ```
- Running dev env
    ```bash
    grunt dev
    ```
